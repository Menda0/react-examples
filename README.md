## How to run

Clone projet

```bash
  git clone https://github.com/Menda0/react-examples
```

Go inside directory

```bash
  cd react-examples
```

Install dependencies

```bash
  npm install // or yarn install
```

Start developing server

```bash
  npm run dev // or yarn dev
```
